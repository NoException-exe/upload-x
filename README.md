## Description

API created using NestJS and PostgreSQL, with functionalities for uploading and downloading ZIP files. Documentation is done with Swagger, making it easier to understand and use the API.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
```
